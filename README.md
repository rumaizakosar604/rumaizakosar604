## 🧠 About Me

I'm an AI researcher passionate about Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and Knowledge Graphs. I enjoy building intelligent AI systems that combine structured knowledge with modern language models to improve reasoning, retrieval, and question answering.

My research focuses on developing privacy-aware RAG frameworks that enhance information retrieval while preserving sensitive data and maintaining high-quality responses.

- 🎓 MS in Artificial Intelligence
- 🔬 AI Researcher — specializing in LLMs, RAG & Knowledge Graphs
- 🧩 Research Focus — Privacy-Preserving AI & Intelligent Information Retrieval
- 💻 Building AI applications using Python, LangChain, FAISS & Neo4j
- 📚 Interested in NLP, Machine Learning, and Generative AI


## 🚀 What I Build

- 🤖 **Privacy-Aware RAG System** — Built an end-to-end Retrieval-Augmented Generation framework with data cleaning, sensitive entity detection, relation extraction, Knowledge Graph construction (Neo4j & NetworkX), Universal Privacy Policy (UPP), hybrid retrieval, vector database integration, LLM-based answer generation, and privacy & utility evaluation.

- 🕸️ **Knowledge Graph Pipelines** — Designed knowledge graphs from unstructured documents using entity and relation extraction to enable structured retrieval and reasoning.

- 🧠 **LLM-Powered Applications** — Developing intelligent applications using Large Language Models for question answering, retrieval, and knowledge-enhanced AI systems.

- 📚 **Natural Language Processing Projects** — Fine-tuned BERT for sentence classification and implemented Word2Vec (Skip-Gram) for learning semantic word embeddings.

- 📊 **Machine Learning Models** — Built and evaluated Decision Tree, Support Vector Machine (SVM), and Ensemble Learning models for classification tasks.
